# 图片目录

请将你的个人照片放在这个目录下。

## 使用方法

1. 将你的照片重命名为 `photo.jpg`
2. 放入 `assets/images/` 目录
3. 照片会自动显示在首页的左侧

## 推荐规格

- 格式：JPG, PNG, WebP
- 尺寸：正方形或接近正方形（如 400x400px 或 600x600px）
- 文件大小：建议小于 2MB

## 样式特点

- 圆角裁切（20px 圆角）
- 固定尺寸：200x200px
- 自适应裁剪（object-fit: cover）
- 边框装饰

## 响应式设计

- 桌面端：照片在左侧，文字在右侧
- 移动端：照片在上方居中，文字在下方左对齐
